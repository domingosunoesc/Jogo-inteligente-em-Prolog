O presente trabalho tem como objetivo desenvolver um sistema inteligente utilizando a linguagem Prolog, aplicando conceitos de Programação Lógica e Sistemas Especialistas. O sistema foi projetado para identificar informações a partir de características fornecidas pelo usuário, utilizando inferência lógica e uma base de conhecimento dinâmica.

O jogo funciona de forma interativa, permitindo que o usuário escolha diferentes categorias, como animais, comidas, desportos e eventos históricos. A partir das características informadas, o sistema tenta identificar o elemento correspondente na base de conhecimento. Caso não consiga encontrar uma resposta, o sistema possui capacidade de aprendizado, adicionando novos conhecimentos dinamicamente durante a execução.
